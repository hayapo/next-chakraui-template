# Template for Next.JS + Chakra UI + TypeScript Project with ESlint + Prettier

## Package install

```bash
npm install
```

## How to run

```bash
yarn run dev
```

## Lint or Format code

### Linting

```bash
# next lint
yarn lint:next

# eslint
yarn lint:eslint
```

### Formatting

```bash
# format with next lint --fix
yarn lint:fix

# format with prettier
yarn format
```

## Features

- Next.js
- Typescript
- Chakra UI
- ESlint
- Prittier
